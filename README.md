# archery-team-maker
A team maker for archery competitions, based on qualification round scores

The input is not currently sanitized, so follow the correct format when writing up the data.
- Each row is a new person
- Each person is of the format SURNAME Name,score
- Leave no empty lines
- Ensure the number of archers is divisible by 3

There is example data in this repo, please refer to it if there is any confusion
